# RUNNING

To run the game:

1. Open the template in GitHub Codespaces.
2. Run `sh start.sh` in the terminal.
3. Wait until the port starts showing up.
4. Go to port `6080` and open `VNC.html`.
5. Press **Connect** and wait for Minecraft to pop up.  
   Happy mining! ⛏️
